# AI Resume Assistant (RAG System)

This repository contains documents used by an AI agent built using Sim AI.

The agent uses Retrieval-Augmented Generation (RAG) to answer questions based on my resume and project information.

## Documents
- Resume of Asit Gaikwad
- Data analyst project descriptions
- Questions and answers for evaluation

## Example Questions
- What projects has Asit Gaikwad completed?
- What tools does Asit use for data analysis?
- What is the accuracy of the loan prediction model?

## Technologies
- RAG (Retrieval Augmented Generation)
- Vector embeddings
- AI Agent workflows
- Sim AI platform
